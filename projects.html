Homography RANSAC

This project involved implementing the RANSAC algorithm to estimate a homography matrix between two sets of points. The algorithm was used to find the best fit homography that maps points from one image to another. The steps involved are:

  1. Use SIFT algorithm to detect and describe keypoints in both images
  2. Match the keypoints between the two images using the SIFT descriptors
  3. Compute the homography matrix using the selected point
  4. Calculate RMS as performance/accuracy metric
  5. Repeat the process for a specified number of iterations (found using RANSAC algorithm) and keep the best homography with the most inliers

Finite Element Method GUI

This project involved creating a graphical user interface (GUI) for the Finite Element Method (FEM) on MATLAB. The GUI allows users to input parameters, visualize the mesh/truss, and analyze the results of FEM simulations. The main features of the GUI include:

  1. Truss/Mesh visualization
  2. Input parameters for FEM simulations:
    1. Force Magnitude and Direction
    2. Link length and nodes
    3. Number of links and nodes
    4. Node and Fixture locations
    5. Young's Modulus and Cross-sectional Area
  3. Visualization of simulation results
  4. User-friendly interface for easy navigation

3-Bit Adder IC Design

In this project, I desgined a 3-bit, 3-input fuller adder using 130nm CMOS technology. The design was implemented using Cadence Virtuoso and included the following features:

  1. NAND gate implementation for each 1-bit adder
  2. Average delay of 9 ns
  3. Average switching power of 0.31 mw
